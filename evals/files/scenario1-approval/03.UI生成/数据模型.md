# 数据模型

## 实体：采购合同（PurchaseContract）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| contract_no | varchar(20) | 合同编号 | 自动生成，格式HT-{年份}-{流水号} |
| title | varchar(200) | 合同标题 | 必填 |
| applicant_id | bigint | 采购员ID | 外键 |
| department_id | bigint | 部门ID | 外键 |
| estimated_amount | decimal(15,2) | 预估金额 | 必填，>0 |
| payment_method | tinyint | 付款方式 | 1=一次性，2=分期 |
| installment_count | int | 分期期数 | payment_method=2时必填 |
| supplier_id | bigint | 供应商ID | 外键，必填 |
| status | tinyint | 合同状态 | 1草稿 2待审批 3审批中 4已签订 5执行中 6已完成 7已过期 8已驳回 |
| contract_date | date | 签订日期 | 签订时填写 |
| expire_date | date | 到期日期 | 必填 |
| created_at | datetime | 创建时间 | 自动 |
| updated_at | datetime | 更新时间 | 自动 |

## 实体：审批记录（ApprovalRecord）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| contract_id | bigint | 合同ID | 外键 |
| approver_id | bigint | 审批人ID | 外键 |
| approval_role | varchar(50) | 审批角色 | 部门经理/总经理/财务总监/法务 |
| result | tinyint | 审批结果 | 1通过 2驳回 |
| opinion | text | 审批意见 | |
| created_at | datetime | 审批时间 | 自动 |

## 实体：供应商（Supplier）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| name | varchar(200) | 供应商名称 | 必填，唯一 |
| contact_person | varchar(100) | 联系人 | |
| phone | varchar(20) | 联系电话 | |
| address | varchar(500) | 地址 | |
| status | tinyint | 状态 | 1启用 2禁用 |
