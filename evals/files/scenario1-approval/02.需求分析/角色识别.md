# 角色识别

| 角色名称 | 角色描述 | 权限范围 |
|---------|---------|---------|
| 采购员 | 发起采购申请，填写物料清单 | 创建申请、查看自己的申请、上传附件 |
| 部门经理 | 审核本部门采购申请 | 审批/驳回本部门申请、查看本部门所有申请 |
| 总经理 | 审批大额合同 | 审批/驳回>=5万申请、查看所有申请 |
| 财务总监 | 审批超大额合同及付款确认 | 审批>=20万申请、确认付款、查看所有合同 |
| 法务专员 | 审核合同法律条款 | 审核>=10万合同的条款、查看合同详情 |
| 系统管理员 | 管理系统配置 | 管理角色权限、审批流程配置、供应商信息维护 |
