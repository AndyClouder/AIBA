# 数据模型

## 实体：员工（Employee）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| employee_no | varchar(20) | 工号 | 必填，唯一 |
| name | varchar(50) | 姓名 | 必填 |
| department | varchar(100) | 部门 | 必填 |
| position | varchar(100) | 职位 | |
| mobile | varchar(20) | 手机号 | 必填，11位 |
| email | varchar(100) | 邮箱 | 必填，邮箱格式 |
| office_location | varchar(200) | 办公地点 | |
| status | tinyint | 在职状态 | 1在职 2离职 |
| created_at | datetime | 创建时间 | 自动 |
| updated_at | datetime | 更新时间 | 自动 |
