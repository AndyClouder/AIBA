# 数据模型

## 实体：考试（Exam）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| exam_no | varchar(20) | 考试编号 | 自动生成，格式KS-{年份}-{流水号} |
| subject | varchar(100) | 考试科目 | 必填 |
| exam_date | datetime | 考试时间 | 必填 |
| location | varchar(200) | 考试地点 | |
| status | tinyint | 状态 | 1未开始 2进行中 3已结束 4成绩已发布 |
| created_by | bigint | 创建人 | 外键 |

## 实体：成绩（Score）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| exam_id | bigint | 考试ID | 外键 |
| student_id | bigint | 学员ID | 外键 |
| student_name | varchar(50) | 学员姓名 | |
| department | varchar(100) | 所属部门 | |
| score | decimal(5,2) | 分数 | 必填，0-100 |
| grade | varchar(10) | 等级 | 优秀/良好/及格/不及格，自动计算 |
| status | tinyint | 状态 | 1待发布 2已发布 3复核中 |
| review_status | tinyint | 复核状态 | 0无 1已申请 2已审核通过 3已审核驳回 |

## 实体：复核申请（ReviewRequest）

| 字段名 | 类型 | 说明 | 约束 |
|-------|------|------|------|
| id | bigint | 主键 | 自增 |
| score_id | bigint | 成绩ID | 外键 |
| reason | text | 复核原因 | 必填 |
| result | tinyint | 审核结果 | 1通过 2驳回 |
| reviewer_id | bigint | 审核人 | 外键 |
| new_score | decimal(5,2) | 修改后分数 | 审核通过时填写 |
